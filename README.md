# Paarsh Edu - Admin Dashboard

An Admin Dashboard for the Paarsh Edu E-Learning Platform developed during internship.

## Features

- Admin Authentication
- Dashboard Analytics
- Course Management
- Category Management
- Student Management
- Enquiry Management
- Placement Management
- Blog Management
- Course Preview & Content Management
- MongoDB Atlas Integration
- Dynamic Data Rendering

## Tech Stack

### Frontend
- EJS
- HTML
- CSS
- JavaScript
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose

## Installation

1. Clone the repository

```bash
git clone https://github.com/TanmayeeDeore/Paarsh-Eduplatform.git
```

2. Navigate to project directory

```bash
cd Paarsh-Eduplatform
```

3. Install dependencies

```bash
npm install
```

4. Create `.env` file

```env
MONGO_URI=your_mongodb_connection_string
PORT=3000
```

5. Run the application

```bash
npm start
```

or

```bash
nodemon server.js
```
